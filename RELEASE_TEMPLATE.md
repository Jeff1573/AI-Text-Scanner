## 🛠️ 安装说明

### Windows
1. 下载 `ai-text-scanner-{{VERSION}} Setup.exe`
2. 双击运行安装程序
3. 按照提示完成安装

### macOS
1. 下载 `ai-text-scanner-{{VERSION}}.dmg`
2. 双击挂载DMG文件
3. 将应用拖拽到Applications文件夹

### Linux
1. 下载对应的包文件
2. 使用包管理器安装：
   ```bash
   # Ubuntu/Debian
   sudo dpkg -i ai-text-scanner_{{VERSION}}_amd64.deb
   
   # CentOS/RHEL
   sudo rpm -i ai-text-scanner-{{VERSION}}.x86_64.rpm
   ```
